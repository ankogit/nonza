# Звуки уведомлений

Используемые файлы:

- `join.wav` — участник вошёл в комнату
- `left.wav` — участник вышел из комнату
- `message.wav` — новое сообщение (для отправителя и получателей)
- `mic on.wav` — микрофон включён (для тебя)
- `mic off.wav` — микрофон выключен (для тебя)

Остальные (`error.wav`) можно подключить позже.
